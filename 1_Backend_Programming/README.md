# 🖥️ Back-End Programming — PHP Logic & Server-Side Processing

This section of my portfolio demonstrates my understanding of back-end programming concepts using PHP. These projects focus on how server-side logic processes data, makes decisions, and generates dynamic content for the user. The examples include conditional processing, looping structures, and secure handling of program inputs.

---

## 🔹 Server-Side Logic and Data Processing

Back-end programming is the foundation of how websites and applications operate behind the scenes. These projects highlight key principles such as:

- Writing server-side code that executes before the page is shown to the user  
- Using variables and conditional checks to make decisions  
- Generating dynamic HTML output based on server-side evaluations  
- Ensuring proper data formatting and secure handling of values  

---

## 🔹 Conditional Structures (Decision Making)

One of the showcased projects evaluates gas prices and determines whether the value falls within a specific range. This demonstrates:

- `if` / `else` logic  
- **Compound conditionals** using logical operators (`&&`, `||`)  
- Cleaner decision-making compared to nested `if` statements  

These techniques are essential for validating data and controlling program flow on the server.

---

## 🔹 Repetition and Loop Processing

Another example uses a `while` loop to generate all odd numbers between 1 and 100. This project reinforces:

- Loop control with `while`  
- Incrementing by custom step values  
- Outputting dynamically generated data  

Looping structures are foundational tools for tasks like processing collections, generating reports, and performing repeated calculations.

---

## 🔐 Security Considerations in Back-End Programming

Although these examples use fixed numeric values, the concepts learned directly apply to real-world secure coding:

- **Input Validation:** Ensuring user-supplied values are numeric and safe  
- **Error Handling:** Preventing unexpected behavior from invalid inputs  
- **Injection Prevention:** Enforcing strict data types to avoid malicious manipulation  
- **Safe Output:** Formatting and escaping dynamic content when needed  

Security awareness is a crucial component of back-end development, even in small exercises.
